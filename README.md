# Equine Interaction Two-Way Hypothesis

This repository contains the de-identified dataset and reproducible analysis script for:

Animal Engagement Shapes Human Emotional Responses In Human–Animal Interactions: A Test of the Two-Way Hypothesis

## Files
- `equine_interaction_reproducible_analysis.py`: main analysis script
- `equine_interaction_dataset.csv`: de-identified study dataset

## Requirements
Python 3.11

Packages:
- pandas
- numpy
- scipy
- statsmodels

## How to reproduce
1. Place `equine_interaction_dataset.csv` in the same directory as `equine_interaction_reproducible_analysis.py`
2. Run:

```bash
python equine_interaction_reproducible_analysis.py
